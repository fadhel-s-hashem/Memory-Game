# Memory Game 🍎 🍇 🍍

A fun, fast-paced browser-based memory matching game. Players must flip hidden cards and find all 6 matching fruit pairs before the 30-second timer runs out.

![the game image](assets/Memory%20Game.png)



## Getting Started

### Play the Game
👉 [Live Deployed Game Link](https://fadhel-s-hashem.github.io/Memory-Game/)

### How to Play

1. **Study the Board**: When the page loads or when you hit **Restart**, the emojis are briefly displayed face-up. Look carefully and memorize their positions!
2. **Start Play**: Click the **start play** button to hide the emojis and kick off the 30-second countdown clock.
3. **Match Pairs**: Click any two squares to reveal their hidden emojis. 
   - If they match, they turn **green** and stay revealed.
   - If they mismatch, they hide themselves automatically after half a second.
4. **Win Condition**: Match all 6 pairs before the clock hits 0 to win the game! Your timer panel will celebrate by turning **lightgreen**.
5. **Loss Condition**: If the countdown hits 0 before you find all pairs, the game ends. The board cards and the timer box background will automatically shift to a **reddish-pink** theme to signal a _Game Over_.



### Installation

No installation required! To play, simply copy the repository to your device and open the `index.html` file in any web browser.

```bash
git clone 
cd Memory-Game
open index.html
```
